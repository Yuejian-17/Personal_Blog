# 如何在 Windows 上搭建舒适的开发环境

## 终端

放弃 CMD 和 PowerShell，直接上 **Windows Terminal** + **WSL2**。配合 oh-my-zsh，效率和美观度直线上升。

## 编辑器

**VS Code** + **Vim 插件**。重度使用 `Ctrl+Shift+P` 和自定义快捷键，尽量避免用鼠标。

## 版本管理

Git + GitHub，另外搭配 `gh` 命令行工具可以直接在终端里创建 Issue、PR，丝般顺滑。

## 其他工具

- **Docker**：无论什么环境，一键拉起，不用担心污染宿主机。
- **Everything**：秒搜文件，比 Windows 自带搜索快 N 倍。
- **PowerToys**：微软官方的效率工具集，尤其推荐 FancyZones 和 Keyboard Manager。
