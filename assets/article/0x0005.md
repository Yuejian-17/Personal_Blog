# 冒泡排序：最经典的排序算法入门

## 什么是冒泡排序？

冒泡排序（Bubble Sort）是最基础、最直观的排序算法之一。它的名字来源于较小的元素会像气泡一样逐渐“浮”到数组的顶端（或较大的元素沉到底部）。

虽然在实际生产中很少直接使用冒泡排序（因为效率较低），但它简洁易懂，是初学者理解**循环**、**比较**和**交换**的绝佳范例。

## 核心思想

冒泡排序重复地遍历要排序的数列，一次比较两个相邻的元素，如果它们的顺序错误就把它们交换过来。遍历数列的工作重复进行，直到没有元素需要交换，也就是该数列已经排序完成。

这个过程就像水里的气泡不断上浮，越大的气泡上升得越快，最终最大（或最小）的元素会被“推”到最后面。

## 算法步骤

给定一个长度为 `n` 的数组 `arr`，算法执行如下：

1. 从第一个元素开始，比较相邻的两个元素 `arr[i]` 和 `arr[i+1]`。
2. 如果 `arr[i] > arr[i+1]`（以升序为例），则交换它们。
3. 继续向后比较，直到数组末尾。此时，最大的元素就已经被放到了正确的位置（数组最后一位）。
4. 重复步骤1~3，但每次遍历的范围都可以减少1（因为最后一个元素已经有序）。
5. 当某次遍历中没有任何交换发生时，说明数组已经有序，算法可以提前终止。

## 动画演示

假设我们有数组 `[5, 3, 8, 6, 2]`，来看看冒泡排序是如何工作的：

**第一轮遍历：**

- 比较 5 和 3：5 > 3，交换 → `[3, 5, 8, 6, 2]`
- 比较 5 和 8：5 < 8，不交换 → `[3, 5, 8, 6, 2]`
- 比较 8 和 6：8 > 6，交换 → `[3, 5, 6, 8, 2]`
- 比较 8 和 2：8 > 2，交换 → `[3, 5, 6, 2, 8]`

**第二轮遍历：**

- 比较 3 和 5：不交换
- 比较 5 和 6：不交换
- 比较 6 和 2：6 > 2，交换 → `[3, 5, 2, 6, 8]`

**第三轮遍历：**

- 比较 3 和 5：不交换
- 比较 5 和 2：5 > 2，交换 → `[3, 2, 5, 6, 8]`

**第四轮遍历：**

- 比较 3 和 2：3 > 2，交换 → `[2, 3, 5, 6, 8]`

**第五轮遍历：** 没有任何交换，排序完成！

## C++ 实现

下面是冒泡排序的 C++ 代码，包含基本的排序函数和优化版本：

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 基本冒泡排序（升序）
void bubbleSort(vector<int>& arr) {
    int n = arr.size();
    for (int i = 0; i < n - 1; i++) {
        // 每一轮遍历将最大元素放到最后
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交换相邻元素
                swap(arr[j], arr[j + 1]);
            }
        }
    }
}

// 优化版：增加标志位，提前结束
void bubbleSortOptimized(vector<int>& arr) {
    int n = arr.size();
    bool swapped;
    for (int i = 0; i < n - 1; i++) {
        swapped = false;
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        // 如果这一轮没有发生交换，说明已经有序，直接退出
        if (!swapped) break;
    }
}

int main() {
    vector<int> arr = {5, 3, 8, 6, 2};
    
    cout << "排序前：";
    for (int x : arr) cout << x << " ";
    cout << endl;
    
    bubbleSortOptimized(arr);
    
    cout << "排序后：";
    for (int x : arr) cout << x << " ";
    cout << endl;
    
    return 0;
}
```

### 代码解析

- `bubbleSort` 函数：外层循环控制总共需要遍历多少轮（最多 `n-1` 轮）。内层循环执行相邻比较和交换，每轮结束后，数组最后 `i` 个元素已经有序。
- `bubbleSortOptimized` 函数：增加了一个 `swapped` 标志，如果在某一轮遍历中没有发生过交换，说明数组已经完全有序，就可以提前结束，减少不必要的比较。这在数组基本有序时能大幅提升性能。
- `main` 函数中演示了如何使用该算法。

## 时间复杂度分析

- **最坏情况**：数组完全逆序，需要 `n-1` 轮遍历，每轮比较 `n-i-1` 次，总比较次数为 `(n-1)+(n-2)+...+1 ≈ n²/2`，所以时间复杂度为 **O(n²)**。
- **最好情况**：数组已经有序，优化后的算法只需一趟遍历（没有交换即退出），时间复杂度为 **O(n)**。
- **平均情况**：O(n²)。
- **空间复杂度**：只使用了常数个额外变量，所以是 **O(1)**，属于原地排序算法。

## 稳定性

冒泡排序是**稳定**的排序算法。当两个元素相等时，我们不会交换它们的位置，所以它们的相对顺序在排序后保持不变。

## 总结

冒泡排序虽然效率不高，但它以极其简洁的逻辑展示了排序的基本原理——比较和交换。它是许多更复杂排序算法（如快速排序、归并排序）的前置知识。理解冒泡排序，就像学骑车时先掌握平衡，之后再学变速和漂移就会容易得多。

实际开发中，对于小规模数据或者基本有序的数据，冒泡排序的优化版本有时也会被采用。
