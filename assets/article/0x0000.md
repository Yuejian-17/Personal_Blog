# 我的博客搭建之旅

## 缘起

一直想拥有一个自己的博客，不受平台限制，能完全掌控样式和内容。

## 技术选型

起初在 Hugo 和 Hexo 之间犹豫，最终决定**挑战自己**，用原生 JavaScript 从零开始搭建。

原因很简单：

- 深入理解 Web 基础
- 完全自由定制
- 体积小，性能极致

## 遇到的坑

1. **路由设计**：SPA 的 hash 路由需要自己处理前进后退，起初总会出现重复渲染的问题，后来通过事件委托和条件判断解决了。
2. **Markdown 渲染**：选择了轻量级的 `marked` 库，几 KB 的体积就能把 Markdown 转成干净的 HTML。

## 总结

现在你看到的这个博客，就是我用原生 JS 一行行写出来的。它不完美，但属于我。
